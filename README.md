# split_phy  

## python version  

`3.9.7`

## install package  

```shell  
pip install pyqt
```

## `.ui` to `.py`  

```shell  
pyuic5 -x UI.ui -o UI.py
```

## run code  

```shell  
cd ./src
python main.py
```

## packaging code  

```shell  
pip install pyinstaller
```

```shell  
pyinstaller -F app.py
```
